## 🛠️ Como Usar

1. Clone o repositório:  

   ```bash
   git clone https://github.com/seu-usuario/nome-do-repositorio.git
   ```

2. Instale as dependências necessárias (se houver):  

   ```bash
   npm install
   ```

3. Execute os testes (dependendo da ferramenta):  

   ```bash
   npm run test
   ```
